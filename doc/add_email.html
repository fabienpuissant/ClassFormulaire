add_email()

Arguments

  • String : The label of the email (Text which will be on the top of the input)
  • String : The name of the email input
  • Boolean : Optionnal true -> the input is reuired , false -> the input is not required. Default value : false
  • String : Optionnal placeholder of the input. Default : no placeholder

Returns

None, the Html code of an email input is generated in PHP
